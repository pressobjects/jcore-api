JCoreOPFConfig

type IJCoreOPFConfiguration = interface(IInterface)

Description

Interface of the global OPF configuration.

Use TJCoreOPFConfiguration class to create a new configuration instance. 

    var
      VConfig: IJCoreOPFConfiguration;
    begin
      VConfig := TJCoreOPFConfiguration.Create;
